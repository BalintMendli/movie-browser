## Usage

### Prerequisites

- node
- npm

### Getting started

Clone the repository to your local machine

```
$ git clone https://github.com/zenott/movie-browser.git
```

Change into the project directory

```
$ cd movie-browser
```

Install dependencies

```
$ npm install
```

Start development server

```
$ npm run start
```

### Run the test suite

```
$ npm run test
```
